# 📊 Finance Analysis Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive Finance Analysis Dashboard built using Microsoft Power BI.

It provides insights into:

- Total Amount
- Total Transactions
- Average Transaction Value
- Total Fees
- Total Tax
- Customer Segments
- Transaction Status
- State-wise Analysis
- Gender Analysis
- Monthly Trends


## 🚀 Features

- Interactive Dashboard
- Dynamic KPI Cards
- Slicers for Year, Occupation and Category
- Drill-through Page
- Dynamic Measures
- Cross-filtering Visuals


## 🛠 Tools Used

- Microsoft Power BI
- DAX
- Power Query
- Excel / CSV
